This blueprint creates an instance and installs nagios 3 server.
##Infra

 2.Azure
 3.Openstack

## Quick Start
 1. Upload the blueprint to calm.
 2. Change Aws provider in overview tab.
 3. Change Credentials DEMO, under credentials tab.
 4. Deploy the blueprint.
